DESCRIPTION:
Here's the Nod Demo for TS M, it contains the first 4 maps. Note that this is a demo and so not everything is "correct".
You need Tiberium Essence 1.6 to correctly play the maps !
Hope you enjoy !
-------------------------------------------------------------------------------------------------------------------------
HOW TO INSTALL/PLAY:
-------------------------------------------------------------------------------------------------------------------------
1. Place the 4 files of the downloaded .rar file in "AppData\Roaming\Command & Conquer 3 Tiberium Wars\Maps\"
(2. More informations on: http://www.cnclabs.com/cnc3/faq.aspx)
3. Launch the game with Tiberium Essence 1.6, go to: Skirmish > Custom Maps
4. Select the map you want to play (and the difficulty)
5. Done !
